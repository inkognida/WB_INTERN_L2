Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
[3 2 3]
```

создается слайс len=3, cap=3
и отправляется в функцию modifySlice().
в срезе хранится указатель на базовый массив, 
modifySlice() изменяет значения исходного среза.
функция modifySlice изменяет исходный срез (указатель не меняется), после append выделяется новая память (меняется указатель)
